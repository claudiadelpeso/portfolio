Compiler

Here is the code for a compiler for a fragment of C++ in Haskell. It includes the Grammar with the parser for a small fragment of C++. It also contains a type checker that produces an annotated abstract syntax tree which is passed to the interpreter that interprets the code. There is also code for a code generator that takes the result of the type checker and produces java class files with JVM code. Finally, it includes an interpreter for a small fragment of haskell (a functional programming language).

Haskell, BNFC

Turtle Graphics

In this project I implemented a copy of Turtle Graphics. It's a program where a turtle carries out the actions you input. Some actions include: move (forward, backwards), rotate, put pen up (stop drawing), put pen down (start drawing)... I implemented the turtle graphics languages as deep embedded language. There are two options when running the program. You can choose to print all the actions the turtle performs (textual interface) or you can choose to draw the actions graphically.

Haskell, HGL library
